# DrawingApp

This is a small drawing program for a desktop environment.

## How to run on Ubuntu:

  sudo apt-get install libsfml-dev
  sudo add-apt-repository ppa:texus/tgui-0.8
  sudo apt-get update
  sudo apt-get install libtgui-dev
  ./build.sh
